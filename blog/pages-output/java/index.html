Java

I've been using Java since early 2000, first for college-related tasks and then for private projects.

Since joining Erudine I have used Java intensively since the legacy codebase had been written in Java, and it involved Spring injection, Java reflection, inversion of control supporting classes that allowed the program to use a functional paradigm and dealing with the Java JAR signing and security mechanism for our license server.

During my work in Eurdine, I was able to improve a process of an executor that had been built around the main tool 500-fold, by using precompiled regular expressions in a piece of code that matched text and was used heavily by the program.

Among others, libraries that I have used include:

  • Guava
  • Goldman Sachs collections
  • JODA Time
  • AspectJ weaver
  • Jide
